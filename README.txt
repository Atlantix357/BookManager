LOCAL BOOK MANAGEMENT APPLICATION
===============================

This application allows you to manage your personal book collection locally on your computer.
No internet connection is required after initial setup.

INSTALLATION & SETUP
-------------------
1. Make sure you have Node.js installed on your computer
   (Download from https://nodejs.org if needed)

2. Double-click "setup.bat" to install required dependencies
   (This only needs to be done once)

RUNNING THE APPLICATION
----------------------
Option 1 (Development Mode):
- Double-click "start_app.bat" to run in development mode

Option 2 (Recommended - Production Mode):
- Double-click "build_app.bat" to create an optimized production build
- Then double-click "run_production.bat" to run the application

If the browser doesn't open automatically, go to:
http://localhost:3500

TROUBLESHOOTING
--------------
If you encounter a blank page:
1. Try using the production mode (Option 2 above)
2. Check that no other application is using port 3500
3. Make sure all dependencies were installed correctly

For any issues, please contact technical support.
